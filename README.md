# CalC
A Command Line based Calculator

#**Operations supported (Only Binary)**
  Solve Infix Notation
  1. Addition
  2. Subtraction
  3. Multiplication
  4. Division
  5. Modulo

#**USAGE**

#Command Line
 
 'cd' into the project root directory

# Linux/Mac
	  ./mvnw compile exec:java

# Windows
	  mvnw.cmd compile exec:java

 To exit, type '**exit**'.
